**@mgremy/core** is a set of utilities, services and helper for Angular application.

# Installation

## Via NPM

```bash
npm i --save-dev @mgremy/core
```

## Via YARN

```bash
yarn add --dev @mgremy/core
```

## Via PNPM

```bash
pnpm i --save-dev @mgremy/core
```
