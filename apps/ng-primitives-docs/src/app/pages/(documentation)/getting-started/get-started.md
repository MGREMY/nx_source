# Get Started

## Installation

`@mgremy/ng-primitives` can be installed via npm. Each components have their secondary entrypoint.

### Via NPM

```bash
npm i --save-dev ng-primitives @mgremy/ng-primitives tailwindcss @tailwindcss/postcss
```

### Via YARN

```bash
yarn i --dev ng-primitives @mgremy/ng-primitives tailwindcss @tailwindcss/postcss
```

### Via PNPM

```bash
pnpm i --save-dev ng-primitives @mgremy/ng-primitives tailwindcss @tailwindcss/postcss
```

## Usage

In your global CSS file, make sure to import both `TailwindCSS` and `@mgremy/ng-primitives`.

```css
@import 'tailwindcss';
@import '@mgremy/ng-primitives/theme/mgnp.css';
```
