# Button

A button is a clickable element that can be used to trigger an action. This primitive enhances the
native button element with improved accessibility and interaction handling for hover, press and
focus.

## Usage

### Default

```html
<button
  ngpButton
  mgnpButton>
  Default button
</button>
```

### Combobox

```html
<button
  ngpComboboxButton
  mgnpButton>
  Combobox button
</button>
```

### Menu

```html
<button
  ngpMenuTrigger
  mgnpButton>
  Menu button
</button>
```
